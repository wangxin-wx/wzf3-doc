$fetch ​

http请求方法。使用了 ofetch 全局公开 $fetch 方法，示例：

ts
// 请求地址最终为http://127.0.0.1:3001/post，超时时间为1000毫秒
$fetch('/exp/post', { method: 'post'，headers: { 'x-wz-timeout': 1000 } })
选项
headers['x-wz-timeout']接口超时。全局接口超时 proxyTimeout
